# 원문자료 인덱스

상위 문서: [[docs/홈택스 문서 지도]], [[docs/index|전체 인덱스]]

## Vault 기준

옵시디언 vault root는 프로젝트 루트 기준이다.

```text
/Users/nox/Documents/홈택스
```

## 주의

- `source/`는 원문 보관소다. 구현자가 바로 읽는 문서는 `docs/`를 우선한다.
- 전산매체의 필드 길이, 레코드 구조, 파일명 규칙은 원문 제출요령을 우선한다.
- 기획서는 서비스 정책과 입력값 매핑의 기준이다.
- source export 파일명에는 Notion UUID가 붙어 있으므로 파일명을 바꾸면 기존 링크가 깨질 수 있다.
