## 추출 품질

- PDF 추출본은 표와 본문이 비교적 잘 유지된다.
- DOC 추출본은 원천세 본문/표 확인에 사용 가능하다.
- HWP 추출본은 `BodyText/Section` 스트림 직접 파싱 결과라 일부 제어 문자열이 섞인다. 그래도 레코드 구성, 길이, 자료구분, 주요 설명 확인에는 사용할 수 있다.
