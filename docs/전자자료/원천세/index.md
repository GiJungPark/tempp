## 원문 기준

원천세 전자파일 생성의 기준 문서는 `source/전산매체`에 보관한 국세청 전산매체 제출요령 DOC/HWP다.

- [[source/전산매체/전산매체/원천세 신고 전산매체 2faae3b52583802b8edec7c6969799e9|source 원천세 신고 전산매체 정리본]]
- [[docs/원문자료/추출본/원천세이행상황신고서_전산매체제출요령_20260102.doc.txt|원천세 DOC 추출본]]
- [[docs/원문자료/추출본/원천세이행상황신고서_전산매체제출요령_20260102.hwp.txt|원천세 HWP 추출본]]

사용자가 정리한 표는 서비스에서 받는 입력값을 전산매체 필드에 매핑하기 위한 기획 자료다. 필드 길이, 파일명, 코드표, 작성 유의사항은 원문 DOC를 우선한다.

## 핵심 규칙

```text
21 Header 레코드                  400 byte
22 환급세액 조정 레코드           200 byte
23 원천징수 명세 및 납부세액      150 byte, 소득코드별 반복
```

대표 소득코드:

```text
사업소득: A25, 가감계 A30
기타소득: A42, 가감계 A40
수정신고: A90 후보
```

파일명:

```text
{일자}C103900.{신고구분상세코드}
예: 20210207C103900.01
```
