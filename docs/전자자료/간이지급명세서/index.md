# 간이지급명세서 전자자료 인덱스

간이지급명세서 전산매체 파일 생성 규격을 소득 유형별로 분리했다.

상위 문서: [[docs/전자자료/index|전자자료 인덱스]], [[docs/홈택스 문서 지도]]

## 문서

- [[docs/전자자료/간이지급명세서/사업소득 간이지급명세서 전산매체|사업소득 간이지급명세서 전산매체]]
  - 자료구분 `50`
  - A/B/C 레코드 각 170 byte
  - 업종코드, 세율, 지급액/세액 집계 규칙
- [[docs/전자자료/간이지급명세서/기타소득 간이지급명세서 전산매체|기타소득 간이지급명세서 전산매체]]
  - 자료구분 `55`
  - A/B/C 레코드 각 170 byte
  - 필요경비, 소득금액, 소득세/지방소득세 계산 규칙

## 공통 구조

```text
A 레코드: 자료제출자 1건
B 레코드: 지급자별 집계, 지급자 수만큼 반복
C 레코드: 소득자별 상세, 지급자별 소득자 수만큼 반복
```

현재 테스트 가정:

```text
제출자 = 지급자 1명
A 1건
B 1건
C N건
```
